demila

demilaMedia是一款基于jquery的跨平台预览插件

事件方式调用

$("#testlink").on("click", function(){
	$.demilamedia.open([
		{href: "medias/test1.jpg", title: "预览1"},
		{href: "medias/test2.jpg", title: "预览2"},
		{href: "medias/test3.jpg", title: "预览3"},
		{href: "medias/test.mp3", title: "预览4"},
		{href: "medias/test.mp4", title: "预览5"}
	]);
	return false;
});

元素注册事件1（图片）

JS脚本：

$(".testlink2").demilamedia();

页面元素：

<a class="testlink2" rel="demilamedia[test]" href="medias/test3.jpg" title="测试图片1"><img src="medias/test1.jpg" alt="测试图片1" /></a>

元素注册事件2（图片、音频、视频）

JS脚本：

$(".testlink3").demilamedia();

页面元素：

<a class="testlink3" rel="demilamedia[test2]" href="medias/test.mp4" title="测试图片1"><img src="medias/test1.jpg" alt="测试图片1" /></a>

元素注册事件3（flash、其他页面ajax、iframe）

JS脚本：

$(".testlink4").demilamedia();

页面元素：

<a class="testlink4" rel="demilamedia[test3]" href="http://player.youku.com/embed/XOTI4MDc4MDY4" title="测试图片1"><img src="medias/test1.jpg" alt="测试图片1" /></a>
